The documentation is available at [szapp-rtd-tutorial.readthedocs.org](https://szapp-rtd-tutorial.readthedocs.org).

## Installation
Install using `pip` from command-line:

```bash
pip install -i https://test.pypi.org/simple/ uq4rw7demo
```
